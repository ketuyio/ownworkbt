What's the Difference

HTML

Hypertext Markup Language

Create the structure

  • Controls the layout of the content
  • Provides structure for web design
  • the fundamental building block of any webpage

CSS

Cascading Style Sheet

Stylize the website

  • Applies style to the webpage elements
  • Targets various screen sizes to make webpages responsive
  • Primarily handles the "look and feel" of the webpage

Javascript

Increasa interactivity

  • Adds interactivity t the webpage
  • Handles complex functions and features
  • Programmatic code which enhances functionality
